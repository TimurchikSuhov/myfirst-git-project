# Руководство по работе с Git

Основные команды системы контроля версий Git.

## Настройка Git

- `git config --global user.name "Ваше Имя"` - установка имени пользователя
- `git config --global user.email "ваш@email.com"` - установка email пользователя
- `git config --list` - просмотр текущих настроек

## Основные команды

- `git init` - создание нового репозитория
- `git status` - проверка состояния файлов
- `git add .` - добавление всех файлов в индекс
- `git add имя_файла` - добавление конкретного файла
- `git commit -m "Сообщение"` - создание коммита
- `git log` - просмотр истории коммитов
- `git log --oneline` - компактный просмотр истории
- `git diff` - просмотр изменений перед коммитом

## Ветвление

- `git branch` - список всех веток
- `git branch имя_ветки` - создание новой ветки
- `git checkout имя_ветки` - переключение на ветку
- `git checkout -b имя_ветки` - создание и переключение на ветку
- `git merge имя_ветки` - слияние веток
- `git branch -d имя_ветки` - удаление ветки
